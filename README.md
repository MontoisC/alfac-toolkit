# ALFAC Toolkit

**Aquatic Literacy For All Children — Task Browser & Session Planner**

**Live** → https://montoisc.github.io/alfac-toolkit/

## What is it?

A PWA for swimming instructors and families — 65 validated aquatic tasks, session planner, terrain mode, SOS emergency numbers.

**2 modes:** Professional · Family   
**5 languages:** EN · FR · NO · PL · DE
